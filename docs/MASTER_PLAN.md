# Final Project Master Plan

## Project title
**Visual Analytics of Bitcoin: Multi-Scale Market Patterns and Cross-Market Signals**

## Time range
2019-01-01 to 2026-04-30

## Case-study windows
1. COVID-19 pandemic: 2020-02-01 to 2020-06-30
2. Russia–Ukraine war: 2022-02-01 to 2022-05-31
3. U.S. election cycle: 2024-09-01 to 2025-01-31
4. U.S.–Iran / Middle East tensions: final dates to be selected after data review

## Core user questions
- How did BTC evolve over the full period?
- Which days form distinct market-state clusters?
- What happened within selected extreme days?
- How do news, narratives, external assets, and prediction markets relate to BTC?

## System layers
### Macro Overview
Use long-term BTC timeline + calendar heatmap + case windows to support temporal navigation.

### Meso Pattern View
Use daily feature vectors and UMAP / t-SNE to discover typical and abnormal daily market states.

### Meso Feature Explanation
Use parallel coordinates to explain the cluster profile of selected market-state groups.

### Micro Detail View
Use selected-day intraday chart + event panel to explain one extreme day in detail.

### Event Context & Backtracking
Show top headlines, narrative context, and link the selected day back to Macro and Meso.
